## Getting Started

### Prerequisites && Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Ahmed-Almutairi/Eid_app
   ```
2. Install packages

  ```sh
    pip install -r requirements.txt
   ```
  

3.  Setup and run  Flask App  

   ```sh
    export FLASK_APP=app.py
   ```

   ```sh
    flask run
   ```
